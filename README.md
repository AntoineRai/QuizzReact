# QUIZZ REACT

## PREREQUIS
NodeJS doit être installé sur votre machine.

## BACK-END
Pour lancer l'API Laravel, dirigez-vous dans le dossier 'backend'.
```bash
cd backend
php artisan laravel
```
## FRONT-END
Pour lancer l'interface graphique avec React, rendez-vous dans le dossier 'frontend'.
```bash
cd frontend
npm start
```
